# Tag Index (ClickUp)

Purpose: keep ClickUp tags consistent, small, and useful.

## Rules
- Prefer **2–4 tags per task**.
- Avoid synonyms (pick one canonical tag).
- Keep the global tag set small: **target 12–15**, hard cap **20**.
- Before suggesting a new tag: check this file; if a close tag exists, reuse it.

## Canonical tags (current)
### Money / Admin
- `bills` — recurring payments/admin payments
- `tax` — taxes, estimated payments, filings
- `insurance` — insurance-related (auto/health/home)

### Car
- `car` — anything car-related
- `dmv` — registration, inspection requirements, DMV paperwork
- `maintenance` — routine upkeep (oil, tires, inspection)
- `warranty` — warranty coverage, claims, eligibility

### Legal / Docs / Ops
- `legal` — NDAs, incorporation, contracts
- `receipt` — reimbursement, expense docs, proof of purchase
- `travel` — trips, bookings, itineraries

### Priority / workflow
- `urgent` — triggers aggressive overdue nagging behavior in our alert script

## Deprecated / don’t use
- (none yet)

## Adding a tag (checklist)
When proposing a new tag, answer:
1) Is it a **category** (good) or a **one-off** (bad)?
2) Will it apply to **5+ tasks/year**?
3) Does it overlap an existing tag?

If no to (1) or (2), don’t add it.
